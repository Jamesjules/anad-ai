# अनाद — Anad.ai

> *Anadi (अनादि) — Without beginning. Without end. Belonging to no one. Available to all.*

---

## What is Anad?

Anad is public AI.

Not owned by a corporation. Not controlled by a government. Not sold to advertisers.

It runs on everyday devices — servers, desktops, laptops, phones — contributed freely by people who believe AI should belong to humanity.

---

## The Problem

Today's AI is:
- Owned by a handful of corporations
- Trained on data taken without consent
- Monetized through surveillance
- Inaccessible to billions of people
- One boardroom decision away from being shut down or changed

---

## The Solution

```
Contribute your device → Earn compute credits
Spend credits → Use AI freely
Network grows → Everyone benefits
```

That's the whole thing. Simple. Functional. Forever.

---

## Core Principles

**1. Privacy by Architecture**
Your queries are processed and immediately forgotten.
Nothing is stored. Nothing is logged. Nothing can be sold.
We cannot betray what we never kept.

**2. No Ownership**
No company owns Anad.
No government controls Anad.
The code is public. The model is public. Everything is public.

**3. Built From Scratch**
No corporate model underneath.
No Llama. No GPT. No hidden dependencies.
Anad's model is trained on consented public data only.
Clean from birth.

**4. Earn What You Use**
Credits are compute time — not money.
Contribute your device → earn credits.
Use Anad → spend credits.
Credits have no monetary value. Ever.

**5. Transparent Always**
Every line of code is public.
Every model update is logged.
Every training data source is documented.
Nothing hidden. Ever.

## Getting Started

### Run a Node

```bash
git clone https://github.com/anad-ai/anad
cd anad
pip install -r requirements.txt
python node.py
```

That's it. You're now part of the network.

### Use Anad

Visit anad.ai or run locally:

```bash
python client.py
```

## Training Data Philosophy

We train only on:
- ✓ Project Gutenberg (70,000 free books)
- ✓ Wikipedia (all languages)
- ✓ ArXiv (scientific papers)
- ✓ Public domain texts
- ✓ Community donated content
- ✓ Open source code

We never train on:
- ✗ User conversations
- ✗ Private data
- ✗ Scraped social media
- ✗ Copyrighted content
- ✗ Anything without consent
